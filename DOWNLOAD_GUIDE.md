# 下载 LocalAIStudio 的详细指南

## 📦 获取方式

由于 GitHub Release 的权限限制，最新的可执行文件通过 GitHub Actions 构建产物（Artifacts）方式提供。

---

## 🚀 方式一：从 GitHub Actions 下载（推荐）

### 步骤 1：访问 Actions 页面
- 点击访问：[项目 Actions 页面](https://github.com/ZA12A230/Windows-bendi-ai-yuancheng/actions)

### 步骤 2：找到成功的构建
- 在页面中找到 **最近的构建**（标签显示为 "Success"）
- 最新的构建会在最顶部，通常是带有 **v1.0.x** 标签的构建
- 点击该构建进入详情页

### 步骤 3：下载 Artifacts
- 在构建详情页向下滚动到 **"Artifacts"** 部分
- 找到名为 **`LocalAIStudio-EXE`** 的文件
- 点击文件名即可下载（下载格式为 ZIP 压缩包）

### 步骤 4：解压文件
- 下载后解压 ZIP 文件
- 里面包含两个文件：
  - `LocalAIStudio.exe`（可直接运行的单文件程序）
  - `LocalAIStudio-win-x64.zip`（压缩包备份）

---

## 🔍 说明

### Artifacts 保留期限
- 构建产物将保留 **30 天**
- 30天后会自动清理，如需长期保存请自行备份

### 最新版本建议
- 建议总是下载最新（最顶部）的构建
- 每个构建对应不同的版本（v1.0.31, v1.0.32 等）
