# Build 文件夹说明

## ❗ 重要提示

此文件夹不包含实际的 EXE 可执行文件！

## 📦 获取真正的 EXE 文件

请通过以下方式获取：

1. 访问 [GitHub Actions 页面](https://github.com/ZA12A230/Windows-bendi-ai-yuancheng/actions)
2. 找到最新的成功构建（绿色标记）
3. 在 "Artifacts" 部分下载 `LocalAIStudio-EXE`
4. 解压 ZIP 文件后获得 `LocalAIStudio.exe`

## 📖 详细下载指南

请查看根目录下的 [DOWNLOAD_GUIDE.md](../DOWNLOAD_GUIDE.md)
