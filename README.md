# AI 本地部署工具箱

一个功能强大的Windows桌面应用程序，用于自动下载、部署和管理本地AI模型，同时提供内网穿透和系统优化功能。

## ✨ 主要功能

### 🤖 AI模型管理 (LM Studio)
- **一键下载**：点击模型卡片即可下载，无需复杂操作
- **支持模型**：
  - GPT-3.5-Turbo (4GB)
  - Llama 3.1 8B (4.7GB) ⭐
  - Qwen 2.5 7B (4.4GB)
  - Phi-3.5 Mini (2.2GB)
  - Mistral 7B (4.1GB)
  - CodeLlama 7B (3.8GB)
  - DeepSeek Coder 6.7B (3.6GB)
  - Yi-1.5 6B (3.5GB)
- **智能检测**：自动检测LM Studio安装状态
- **自动部署**：下载完成后自动集成到LM Studio

### 🌐 内网穿透 (frp)
- **自定义服务器地址**：支持配置用户自己的frp服务器
- **认证功能**：
  - 自定义用户名
  - 自定义密码
  - 安全的身份验证
- **连接测试**：配置前可测试连接是否可用
- **自动重连**：网络断开后自动重连
- **公网访问**：支持从外网远程访问本地AI服务

### ⚡ 性能监控
- 实时监控CPU、内存、GPU使用率
- 可配置性能阈值
- 系统负载过高时自动降低AI资源占用
- 性能恢复后自动恢复

### 💻 Windows系统优化
- 开机自启动
- 后台运行（最小化到系统托盘）
- 修改开始菜单关机键为息屏
- 真正关机需要按电源键

## 📖 使用指南

### 模型管理

1. **首次使用**：应用会检测LM Studio是否安装
2. **下载模型**：
   - 直接点击模型卡片即可开始下载
   - 下载进度实时显示
   - 可同时下载多个模型
3. **启动服务**：
   - 下载完成后，点击「打开LM Studio」
   - 在LM Studio中选择模型并加载
   - 启用Server功能

### 内网穿透配置

1. 进入「设置」页面
2. 在「内网穿透配置」中填写：
   - **服务器地址**：你的frp服务器域名或IP
   - **服务器端口**：frp服务端口（默认7000）
   - **认证用户名**：frp认证用户名
   - **认证密码**：frp认证密码
   - **本地端口**：LM Studio的API端口（默认1234）
3. 点击「测试连接」验证配置
4. 开启「启用内网穿透」开关
5. 点击「保存设置」

### 系统设置

- **开机自启动**：系统启动时自动运行应用
- **后台运行**：关闭窗口时最小化到托盘
- **关机键息屏**：关机改为息屏，需按电源键真正关机
- **性能阈值**：自动调节的资源使用上限

## 🔧 配置项说明

### 内网穿透配置

| 配置项 | 说明 | 示例 |
|--------|------|------|
| 服务器地址 | frp服务器域名或IP | frp.example.com |
| 服务器端口 | frp服务端口 | 7000 |
| 认证用户名 | 服务器认证用户名 | myuser |
| 认证密码 | 服务器认证密码 | mypass |
| 本地端口 | LM Studio API端口 | 1234 |

### 系统配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| autoStart | 开机自启动 | false |
| runInBackground | 后台运行 | true |
| enableTunnel | 启用内网穿透 | false |
| performanceThreshold | 性能阈值 | 80% |
| shutdownToSleep | 关机键改息屏 | false |

## ⚠️ 重要提示

### 关于AI模型
- 本应用使用 **LM Studio** 作为本地AI运行时
- 需要先下载安装 [LM Studio](https://lmstudio.ai/download)
- 模型下载后需在LM Studio中手动加载

### 关于内网穿透
- 需要自行搭建或租用frp服务器
- 推荐使用带认证的frp配置确保安全
- 内网穿透地址仅在服务器在线时可用

### 关于系统要求
- Windows 10/11
- 建议16GB+内存以获得最佳体验
- NVIDIA显卡可获得更好的推理性能

## 🔒 安全建议

1. **内网穿透认证**：务必设置frp认证用户名和密码
2. **端口安全**：不要将非必要端口暴露到公网
3. **访问控制**：建议配合防火墙使用
4. **数据安全**：敏感操作建议使用VPN

## 🤝 技术支持

如遇到问题，请检查：
1. LM Studio是否正确安装
2. frp服务器是否在线
3. 防火墙设置是否正确
4. 端口是否被占用
